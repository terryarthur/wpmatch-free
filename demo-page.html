  1. Upload the plugin to /wp-content/plugins/wpmatch-free/

  3. Go to WP Match in admin menu for settings
  4. Create pages and add shortcodes or blocks as shown below

üîß Available Shortcodes

Profile Editor

[wpmf_profile_edit]

Allows logged-in users to create and edit their dating profile with gender, region, headline, and bio fields.

Usage: Add to a page called "Edit Profile"

Search Form

[wpmf_search_form]

Provides a search form with filters for region, age range, and photo requirements.

Usage: Add to your main dating page

Search Results

[wpmf_search_results]

Displays search results based on form parameters, showing profile cards with headlines and regions.

Usage: Add below the search form or on a separate results page

üöÄ REST API Endpoints

Profiles List

GET /wp-json/wpmatch-free/v1/profiles

Parameters: page, per_page, age_min, age_max, region, verified, has_photo

Returns paginated list of active dating profiles with filtering options.

Profile Detail

GET /wp-json/wpmatch-free/v1/profiles/{user_id}

Returns detailed information about a specific user's dating profile.

My Matches

GET /wp-json/wpmatch-free/v1/matches/me

Auth: Required

Returns potential matches for the current user based on region compatibility.

Likes Received

GET /wp-json/wpmatch-free/v1/likes/me

Auth: Required

Shows users who have liked the current user's profile.

üõ°Ô∏è Safety & Moderation Features

Rate Limiting

Messages: 20 per day (configurable)
Likes: 50 per day (configurable)

Prevents spam and abuse through per-user daily limits.

Word Filter

Comma-separated list of blocked words in messages.

Configure in WP Match > Settings > Moderation & Safety

Photo Moderation

Pre-approval: Photos need approval before visible
Post-approval: Photos visible immediately, reviewed later

Blocking System

Users can block each other. Blocked users:

  • Don't appear in search results
  • Cannot view each other's profiles
  • Cannot send messages
